# üíé Atomic UI Kit (HcKit)

The Atomic UI Kit represents the "Modern" theme component layer. These are low-level, high-performance components styled with Tailwind CSS, designed to be used both inside architectural components and as standalone elements in your admin views.

---

## üìë Component Categories

### 1. [General Components](./general.md)
Foundational elements like buttons, badges, and user avatars.
- `HcButton`, `HcButtonGroup`, `HcBadge`, `HcAvatar`.

### 2. [Form Elements](./forms.md)
The core suite for user input and data entry.
- `HcInput`, `HcSelect`, `HcCheckbox`, `HcSwitch`, `HcRadioGroup`, `HcFileUpload`.

### 3. [Feedback & Loaders](./feedback.md)
Visual cues for status and data loading.
- `HcAlert`, `HcToast`, `HcSpinner`, `HcProgress`, `HcBadgeContainer`.

### 4. [Overlays & Tooltips](./overlays.md)
Floating and contextual interaction layers.
- `HcModal`, `HcDrawer`, `HcTooltip`, `HcPopover`, `HcDropdown`.

### 5. [Navigation & Groups](./navigation.md)
Logical grouping and directional controls.
- `HcTabs`, `HcPagination`, `HcBreadcrumbs`.

### 6. [Data & Layout](./data-layout.md)
Presentation blocks and data containers.
- `HcCard`, `HcTable`, `HcStatCard`, `HcPlaceholder`.

---

## üé® Theme Philosophy
While the legacy components use Bootstrap/Vanilla CSS, the **HcKit** suite is built for the **Modern Theme** and utilizes:
- **Vue 3 Composition API**
- **Tailwind CSS Utility classes**
- **Scoped CSS** for complex component logic (Selection alignment, etc.)
