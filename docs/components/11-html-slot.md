# HtmlSlot

A simple layout utility component that wraps content in a full-width container.

## 🚀 Import

```javascript
import { HtmlSlot } from "@hashtagcms/admin-ui-kit/themes/modern/components";
```

## 📦 Slots

| Slot Name | Description                                           |
| :-------- | :---------------------------------------------------- |
| `default` | The HTML content or component to be rendered within the slot. |

## 📝 Usage Example

```html
<html-slot>
  <div class="custom-content">
    <h1>Page Header</h1>
    <p>This is wrapped in an HtmlSlot.</p>
  </div>
</html-slot>
```
